## About

I build backend systems where the boring parts — auth, permissions, data integrity, query performance — are treated as first-class engineering problems, not afterthoughts. My day-to-day is Django and DRF: modeling domains cleanly, designing REST APIs that are predictable to consume, and making sure the system underneath still holds up as data and traffic grow.

Recent focus areas:
- Multi-role authorization systems (JWT, refresh-token flows, RBAC)
- API design that stays consistent across a growing codebase
- Query and ORM performance under real usage patterns
- Bringing that same rigor to a matching React/TypeScript frontend when a project needs one

Currently based in Dhaka, Bangladesh, working across backend-heavy freelance projects and personal systems built to production standards.
